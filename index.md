# 大语言模型预训练技术演进

**从 Scaling Law 到数据配方与分布式训练**

这是一本关于大语言模型预训练技术演进的中文技术书。它按时间线梳理从词向量、Transformer、GPT/BERT/T5，到 Scaling Law、Chinchilla、数据配方、分布式训练、MoE、长上下文、LLaMA 3 与 DeepSeek-V3 的关键技术变化。

本书重点回答一个问题：大模型预训练为什么会从“把模型和数据做大”演进成一套围绕数据质量、计算预算、训练系统、架构选择和目标函数共同优化的工程科学？

## 本书覆盖

- 2013-2017：词向量、RNN/LSTM、Seq2Seq、Attention 与预训练思想的前史。
- 2017-2020：Transformer、GPT、BERT、T5/BART 与预训练范式成型。
- 2020-2022：Scaling Law、GPT-3、Chinchilla 与计算最优训练。
- 2022-2024：数据配方、开源模型、3D 并行、FlashAttention、训练系统优化。
- 2024-2026：MoE、长上下文、DeepSeek-V3、LLaMA 3、推理模型与未来预训练趋势。
